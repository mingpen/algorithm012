# 33. 搜索旋转排序数组
细分该问题时，
应考虑到mid位置(index)是不变，
是数据变化 nums[mid] 处于不同的段  
第一层分析单调性  
第二层分析target的预期区间
## 全单调
nums[end-1] >= nums[start]

### target > nums[mid]

- ：start=mid+1

### target < nums[mid]

- :  end=mid

## nums[mid] 在上段
nums[mid] >= nums[start]

### target > nums[mid]

- :  start=mid+1

### target < nums[mid]

- target >= nums[start]

	- :  end=mid

- target < nums[start]

	- ：start=mid+1

## nums[mid] 在下段
nums[mid] < nums[start]

### target > nums[mid]

- target >= nums[start]

	- :  end=mid

- target < nums[start]

	- ：start=mid+1

### target < nums[mid]

- :  end=mid
