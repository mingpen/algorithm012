## 培养一个人，比找一个合适的人难

对于我们这种普通人，就必须刻意练习，但是能够找对方法并不断坚持的人比例并不大；培养一个人是要花很多很多时间。  

放弃一件事件的理由可以找很多。坚持一件事情就要有梦想，通俗的说就是欲望强烈。  
培养自己兴趣，看看别人怎么做的（不能闭门造车），吸收别人智慧。
